絡 get entangled

糸 (string) + 各 (each) Same-ON
★★☆☆☆
12 strokes

Onyomi

RAKU

Each ROCK and roller who got entangled with Courtney Love, wound up committing suicide.

Mnemonic

Each string gets entangled with the other string.

Kunyomi

( を ) から＊む to get entangled in (a problem, a crime)
★☆☆☆☆
( が ) から＊まる something gets entangled (unlike からむ、　からまる tends to be used only literally)
★☆☆☆☆

Jukugo

連絡(れんらく) する

inform ★★★★★
連 (take with / inform of) + 絡 (get entangled) = 連絡 (inform)

to inform, to let someone know. Like, "Let me know if you're coming. Call me when you get there," kind of thing.
